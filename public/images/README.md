# Images Directory

This directory contains images for the website. If images are missing, the website will automatically show gradient fallback backgrounds.

## Directory Structure:
- `/images/tours/` - Tour images
- `/images/attractions/` - Attraction images  
- `/images/testimonials/` - Testimonial user photos

## Adding Images:
1. Place images in the appropriate subdirectory
2. Use descriptive filenames (e.g., `accra-city-tour.jpg`)
3. Recommended formats: JPG, PNG, WebP
4. Recommended sizes:
   - Tour featured images: 1200x800px
   - Gallery images: 1920x1080px
   - Testimonial photos: 400x400px

The website will automatically handle missing images with beautiful gradient fallbacks.
